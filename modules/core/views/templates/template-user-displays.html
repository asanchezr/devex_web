User Displays

Templates for how to display users around the site

User Cards

Use a user card when the view should enable the user to remove the card from view, for example:
  • In Admin view on a project page, when the user should have the ability to remove a user card from the project
  • In Admin view on a company members list, when the user should have the ability to remove a member from the company

User card in a Company profile members list (Admin view)

The card should contain
  • User profile image
  • Username
  • A label list of the user's self-claimed capabilties, followed by skills.

User Avatars

Use a user avatar when the view should not have any options to remove the user avatar.
These should look just like user cards, except they don't have a background or a button, because you can't do anything with them! They're just there for information, for example:
  • On an opportunity page, to show the user who has been assigned to the opportunity
  • On a project or program page in non-admin view, to simply see the people who are members of the company
